# SCT_WD_3 — QuizTime! Interactive Quiz Game

A browser-based interactive quiz game built with pure HTML, CSS, and JavaScript that displays questions, collects answers, and provides a detailed score at the end.

## Task Description

TASK 03
INTERACTIVE QUIZ GAME

Develop an interactive multiple-choice quiz game that displays questions, collects answers, and provides a score at the end.
You can also add different types of questions like Single or Multi Select, Fill in the Blanks or more.

## Features

- Multiple choice (single select) and fill-in-the-blank question types
- 4 quiz categories — General Knowledge, Science, Technology, History
- 3 difficulty levels — Easy, Medium, Hard with adjustable countdown timer
- Fixed timer bar per question (30s / 20s / 12s based on difficulty)
- Speed bonus points for answering quickly
- Randomised question and option order on every round
- Answer explanations shown after each submission
- Animated results screen with score ring, accuracy, and average time stats
- Gaming-style title with neon glow animation using Press Start 2P font
- Fully responsive layout for desktop and mobile

## Tech Stack

- **HTML5**
- **CSS3** (custom properties, grid, flexbox, keyframe animations, neon glow effects)
- **JavaScript** (game logic, timer, DOM manipulation, score calculation)
- **Google Fonts** — Press Start 2P
